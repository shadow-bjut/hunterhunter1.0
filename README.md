## 🗡️ 游戏概述

玩家从两名猎人角色中选择一位，与强大的 Boss 展开对决。通过灵活的移动、精准的攻击与技能释放击败 Boss，在最短时间内保留最多血量，争取最高分并登上排行榜。

## 🚀 快速开始

### 依赖环境

- Windows 操作系统
- Visual Studio（推荐 2019 或更高）
- [EasyX 图形库](https://easyx.cn/)

### 编译运行

```bash
# 1. 克隆仓库
git clone https://github.com/yourname/hunters-duel.git

# 2. 用 Visual Studio 打开项目，确认已安装 EasyX

# 3. 将所需图片资源放置于可执行文件同目录下：
#    Hunter-X-Hunter.jpg  /  game_background.jpg
#    qy_left_resized.jpg  /  qy_right_resized.jpg  /  ...等角色图片

# 4. 编译并运行
```

> ⚠️ 图片资源文件需自行准备并与 `.exe` 放在同一目录。

## 🎮 操作说明

### 移动

| 按键 | 功能 |
|------|------|
| `A` | 向左移动 |
| `D` | 向右移动 |
| `F` | 向右快速冲刺 |
| `W` | 跳跃（支持**二段跳**） |
| `S` *(落地时)* | 蹲下（受弹伤害减半） |

### 战斗

| 按键 | 功能 |
|------|------|
| `J` | 普通攻击（近身，冷却 0.5 秒） |
| `1` | 释放技能一 |
| `2` | 释放技能二（大招） |
| `3` | 释放技能三 |

### 系统

| 按键 | 功能 |
|------|------|
| `R` | 胜利后保存成绩 / 失败后重新开始 |
| `ESC` | 退出游戏 |

## ⚔️ 战斗系统

### 普通攻击

按 `J` 键对 Boss 发动近身攻击，造成 **8 点固定伤害**，有效距离 **150px**，是最基础的输出手段。

### 技能系统

每个技能均有**前摇时间**（释放动作期间无法移动），技能命中需在 **300px** 范围内。

#### 奇犽技能表

| 技能 | 键位 | 伤害 | 前摇 | 冷却 |
|------|------|------|------|------|
| 技能一 | `1` | 25 | 短（20帧）| 3 秒 |
| 技能二 | `2` | 45 | 长（40帧）| 5 秒 |
| 技能三 | `3` | 15 | 极短（10帧）| 2 秒 |

#### 小杰技能表

| 技能 | 键位 | 伤害 | 前摇 | 冷却 |
|------|------|------|------|------|
| 技能一 | `1` | 20 | 短（15帧）| 2.5 秒 |
| 技能二 | `2` | 60 | 极长（50帧）| 6 秒 |
| 技能三 | `3` | 30 | 中（25帧）| 3.3 秒 |

> 💡 **技巧**：蹲下状态（`S`）可使受到的弹幕伤害减半，合理利用可大幅提升生存能力。

## 👾 Boss 机制

Boss 随血量减少共分三个阶段，攻击模式逐步升级：

```
HP 100% ──────────────── 60% ──────────── 30% ──── 0%
   │                      │                │
阶段 0                  阶段 1           阶段 2
单发直线弹            三连散射弹         追踪弹
                    （上 / 中 / 下）   （直接瞄准玩家）
```

| 阶段 | 触发条件 | 弹幕模式 | 效果 |
|------|---------|---------|------|
| 初始 | 满血 | 单发直线 | 攻击节律平稳 |
| 阶段一 | 血量 ≤ 60% | 三连散射 | 攻击间隔缩短，覆盖面更广 |
| 阶段二 | 血量 ≤ 30% | 追踪弹 | 攻击间隔大幅缩短，弹幕直接锁定玩家位置 |

> ⚠️ 每进入新阶段，Boss 的攻击间隔都会进一步压缩，节奏明显加快，请做好心理准备。

## 🏆 计分规则

游戏胜利后按以下公式结算最终得分：

$$
\text{得分} = 1000 + \text{剩余血量} \times 10 - \max(0,\ \text{用时秒数} - 30) \times 5
$$

| 项目 | 说明 |
|------|------|
| 基础分 | 1000 分 |
| 血量奖励 | 每点剩余 HP +10 分 |
| 超时惩罚 | 超过 30 秒后，每秒 -5 分 |
| 失败 | 得分为 0 |

> 🎯 **目标**：快速击败 Boss 且尽可能少掉血，是取得高分的关键！

## 📋 排行榜

- 胜利后按 `R` 键可输入昵称保存成绩
- 昵称支持 **A–Z、0–9、下划线**，最多 **21 个字符**
- 排行榜保留前 **5 名**，成绩持久化存储于 `scores.dat`

```
🥇  PlayerOne   1480
🥈  Hunter_X    1350
🥉  GonFreecs   1200
    KillUaFan    980
    Anonymous    760
```
